Kifu-recorder

This is an Android application that aims to watch and record go (igo, baduk, weiqi) games using the camera of a smartphone. The games are stored in the SGF format.

Testing results

  • It's important to store which version of the code and which hypotheses were used to run each test

Images description

  • angle (almost orthogonal, 45, acute, very acute)
  • light (optimal, presence of shadows)
  • number of stones
  • size of board (9x9, 13x13, 19x19)

Results

  • True positives: number of correctly detected stones
  • False positives: number of incorrectly detected stones
  • False negatives: number of stones that should have been detected but were not
